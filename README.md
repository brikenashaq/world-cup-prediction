# 🏆 FIFA World Cup 2026 Predictor

A machine learning web app that predicts World Cup 2026 match outcomes and simulates the full tournament using Monte Carlo simulation — updated with live match data from the API.

**🌐 Live App → [worldcup2026-predictor.streamlit.app](https://worldcup2026-predictor.streamlit.app)**

---

## What it does

- **Predicts match outcomes** — win/draw/loss probabilities for any two teams
- **Simulates the full tournament** — runs 10,000 simulations to estimate each team's probability of winning the World Cup
- **Live tournament tracker** — group standings, bracket, top scorers, all updated from the API
- **Remaining matches** — predictions for every upcoming knockout match
- **Watch live** — links to free streams for every match

---

## How it works

### 1. Data
- **Historical matches** — 15,000+ international matches from 1993–2026 (Kaggle)
- **Elo ratings** — pre-match team strength ratings merged per match (Kaggle)
- **Live data** — current tournament results via [football-data.org](https://football-data.org) API

### 2. Features
For each match the model sees 8 features:

| Feature | What it captures |
|---|---|
| `elo_diff` | Overall team strength difference |
| `home_form` | Weighted points per game (last 10 matches) |
| `away_form` | Weighted points per game (last 10 matches) |
| `home_avg_goals` | Rolling average goals scored |
| `away_avg_goals` | Rolling average goals scored |
| `home_avg_conceded` | Rolling average goals conceded |
| `away_avg_conceded` | Rolling average goals conceded |
| `neutral` | Whether match is at a neutral venue |

Form and goals are **tournament-weighted** — a World Cup match counts more than a friendly.

### 3. Model
**XGBoost** multiclass classifier predicting Home Win / Draw / Away Win.
- Trained on 15,000+ matches with time-based split (no data leakage)
- Calibrated with isotonic regression so probabilities are reliable
- ~56% accuracy on 3-class prediction (random baseline = 33%)

### 4. Live data blending
When you run `python src/live_data.py`, the app fetches current World Cup results and blends them with historical stats:

```
final_form = (live_weight × tournament_form) + (1 - live_weight) × historical_form
```

Live weight increases with games played — after 6 tournament games, live data counts for 88% of the rating.

### 5. Tournament simulation
- Simulates the full 48-team group stage (round-robin per group)
- Top 2 from each group + best 8 third-place teams advance (32 teams)
- Knockout rounds simulated with no draws (extra time modelled as 50/50 split)
- Run 10,000 times → win count per team → probability table

## Run locally

```bash
# 1. Clone the repo
git clone https://github.com/brikenashaq/world-cup-prediction.git
cd world-cup-prediction

# 2. Create virtual environment
python -m venv .venv
source .venv/bin/activate  # Mac/Linux

# 3. Install dependencies
pip install -r requirements.txt

# 4. Download raw data from Kaggle and place in data/raw/
#    results.csv:    https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017
#    eloratings.csv: https://www.kaggle.com/datasets/tadhgfitzgerald/fifa-international-soccer-mens-ranking-1993now

# 5. Build features (run once)
python src/features.py

# 6. Train model (run once)
python src/train.py

# 7. Fetch live tournament data (run anytime to update)
python src/live_data.py
python src/tournament_data.py

# 8. Launch app
streamlit run app.py
```
